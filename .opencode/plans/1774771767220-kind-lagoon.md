### 2. Add interactive 3D test script
Replace or add alongside existing test:
```nix
scripts.test-plot-3d.exec = ''
  octave --eval "
    graphics_toolkit('qt');  % Use Qt for interactive 3D
    [x, y] = meshgrid(-2:0.2:2);
    z = x .* exp(-x.^2 - y.^2);
    surf(x, y, z);
    title('Interactive 3D Plot - Drag to rotate');
    printf('Interactive window opened. Close it when done.\n');
  "
'';
```

### 3. Keep gnuplot as non-interactive fallback
For headless/CI environments, `gnuplot` + file output remains useful.
